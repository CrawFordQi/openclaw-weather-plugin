# 加法插件

OpenClaw 平台的加法运算插件，支持通过消息和命令两种方式执行加法运算。

## 功能特点

- **消息处理**：识别包含"加"、"加法"或"+"的消息，自动提取数字并计算总和
- **命令支持**：通过 `add` 命令执行加法运算
- **灵活输入**：支持多个数字的连续加法运算
- **中文大写**：结果同时显示阿拉伯数字和中文大写数字
- **错误提示**：当输入不符合要求时会给出友好的提示

## 安装

```bash
# 本地安装
openclaw plugin install ./add-plugin

# 或从 npm 安装（发布后）
openclaw plugin install @openclaw/add-plugin
```

## 使用方法

### 消息方式

发送包含加法运算的消息，例如：
- `1+2`
- `3加4`
- `5+6+7`

### 命令方式

使用 `add` 命令，例如：
- `add 1 2`
- `add 3 4 5`

## 示例

**输入：**
```
1+2+3
```

**输出：**
```
计算结果：陆 (6)
```

**输入：**
```
add 10 20 30
```

**输出：**
```
计算结果：陆拾 (60)
```

## 配置

该插件无需特殊配置，直接安装即可使用。

## 开发

```bash
# 进入插件目录
cd add-plugin

# 本地测试
openclaw dev
```
