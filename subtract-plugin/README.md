# 减法插件

OpenClaw 平台的减法运算插件，支持通过消息和命令两种方式执行减法运算。

## 功能特点

- **消息处理**：识别包含"减"、"减法"或"-"的消息，自动提取数字并计算结果
- **命令支持**：通过 `subtract` 命令执行减法运算
- **灵活输入**：支持多个数字的连续减法运算
- **中文大写**：结果同时显示阿拉伯数字和中文大写数字（支持负数）
- **错误提示**：当输入不符合要求时会给出友好的提示

## 安装

```bash
# 本地安装
openclaw plugin install ./subtract-plugin

# 或从 npm 安装（发布后）
openclaw plugin install @openclaw/subtract-plugin
```

## 使用方法

### 消息方式

发送包含减法运算的消息，例如：
- `5-3`
- `10减2`
- `20-5-3`

### 命令方式

使用 `subtract` 命令，例如：
- `subtract 5 3`
- `subtract 20 5 3`

## 示例

**输入：**
```
10-3
```

**输出：**
```
计算结果：柒 (7)
```

**输入：**
```
subtract 20 5 3
```

**输出：**
```
计算结果：拾贰 (12)
```

**负数示例：**
```
subtract 5 10
```

**输出：**
```
计算结果：负伍 (-5)
```

## 配置

该插件无需特殊配置，直接安装即可使用。

## 开发

```bash
# 进入插件目录
cd subtract-plugin

# 本地测试
openclaw dev
```
